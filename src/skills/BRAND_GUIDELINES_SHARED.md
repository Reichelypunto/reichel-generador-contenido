# 🎯 BRAND GUIDELINES COMPARTIDAS
## Sistema B-Roll (content-plan) + Sistema Carruseles (GPT)

Ambas IAs son **hermanas arquitectónicas** que comparten framework, tono, voz, y validación. Usa esta guía para entender por qué ambas sistemas se sienten cohesionados y cómo usarlas en conjunto.

---

## 1. IDENTIDAD COMPARTIDA

### Nombre & Posicionamiento

**Sistema B-Roll** = Generador de estrategia de contenido de 30 días con scripts multipropósito
**Sistema Carruseles** = Generador de Instagram carousels de alta conversión

**Juntos = Solución completa de contenido viral para creadores, coaches, vendedores**

### What They Do Together

```
B-Roll Strategy (scripts)
    ↓
    ├─→ Scripts para reels/videos
    ├─→ Scripts para lives
    ├─→ Scripts para captions
    └─→ **Invoca Carruseles** para cada script
        
Carruseles (Instagram)
    ↓
    ├─→ Carousels de educación
    ├─→ Carousels de transformación
    ├─→ Carousels de revelación
    └─→ Puede usarse standalone o con B-Roll

**Total Output = Scripts + Carousels = Contenido para 30 días sin repetición**
```

## 2. ARQUITECTURA PSICOLÓGICA COMPARTIDA

### 4 Viral Motors (identical)

| Motor | Descripción | En el usuario | Ejemplo de activación |
|-------|-------------|-------------|----------------------|
| **Aspiración** | "Quiero ser/tener esto" | Deseo de transformación, status, logro | "Cuando dejé de vender y empecé a educar, mis conversiones subieron 300%" |
| **Educación** | "No sabía esto" | Curiosidad, necesidad de aprender | "7 cosas que tu cliente piensa pero no dice" |
| **Impacto** | "Esto me afecta directamente" | Miedo, urgencia, pain | "Estás usando la misma estrategia hace 3 años y preguntas por qué no creces" |
| **Reflejo** | "Esto soy yo" | Identificación, confesión, validación | "Si estás educando sin ofrecer, probablemente también regalas tu experiencia" |

**Uso:** Identifica cuál motor es dominante en tu contenido. Eso define el emotional weight y el estilo de ejecución.

---

### 10 Viral Hook Formulas (identical)

Ambos sistemas usan LAS MISMAS 10 fórmulas. Pueden variar aplicación, pero el framework es idéntico:

| # | Fórmula | Patrón Narrativo | Motor | Uso |
|---|---------|-----------------|-------|-----|
| 1 | **Cotilla** | Contraste | Impacto | Challenging conventional wisdom |
| 2 | **Fórmula** | Punto de Vista | Educación | Teaching frameworks/rules |
| 3 | **Negativo** | Error | Impacto | Attacking mindset/habits |
| 4 | **Pregunta** | Punto de Vista | Curiosidad | Opening alternatives |
| 5 | **Regla de 3** | Lista | Impacto | Multi-factor problems |
| 6 | **Contracorriente** | Contraste | Aspiración | Challenging status quo |
| 7 | **Si Tú** | Confesión | Reflejo | Self-recognition patterns |
| 8 | **Resultado** | Hábito | Aspiración | Proof via transformation |
| 9 | **1 Minuto** | Confesión | Educación | Secret revealed + authority |
| 10 | **La Única Cosa** | Punto de Vista | Impacto | Simplification of complexity |

**Implicación:** Si usas `fórmula=Cotilla` en B-Roll, puedes amplificarla con un carousel también usando Cotilla. Mismo hook, distinto medium.

---

### 3 Execution Styles (identical)

Ambos sistemas usan los mismos 3 estilos de ejecución:

#### NEGATIVO (Attack → Correction → Transformation)
- **Cuándo usar:** Content sobre errores, mindset, hábitos destructivos
- **Perfil:** Coach, transformación, cambio de paradigma
- **Tone:** Direct, uncomfortable, sharp (no harsh)
- **Estructura:** Error → Consequence → Why they do it → Correction → Proof → Transformation
- **Emotional motor:** Impacto (dolor primario)
- **Ejemplos:**
  - B-Roll: "Tu problema no es vender. Es que nadie entiende por qué escucharte."
  - Carousel: Mismo hook, 10 slides desarrollando la corrección

#### INFO SECRETA (Reveal → Mechanism → Application → Result)
- **Cuándo usar:** Contenido sobre revelaciones, hidden knowledge, ventaja competitiva
- **Perfil:** Vendedor, educador, posicionamiento como autoridad
- **Tone:** Confidential, like you're telling them something competitors don't want known
- **Estructura:** Setup → Surprise (el secreto) → Explanation → Application → Result
- **Emotional motor:** Educación (curiosidad satisfecha)
- **Ejemplos:**
  - B-Roll: "Tu cliente no compra por el producto. Compra por la conversación en su cabeza."
  - Carousel: Mismo revelación, 10 slides con mecanismo + aplicación

#### CONTROVERSIAL (Challenge → Evidence → Alternative → Invitation)
- **Cuándo usar:** Contenido que desafía creencias, POV contrarian, stand-out
- **Perfil:** Creador, experto, thought leader
- **Tone:** Provocative, intellectual, bold (but not for shock—for clarity)
- **Estructura:** Belief (widely held) → Challenge → Evidence → Alternative → Invitation to think differently
- **Emotional motor:** Aspiración (or Impacto, depending on intensity)
- **Ejemplos:**
  - B-Roll: "El engagement no vende. Ni seguidores. Ni algoritmo."
  - Carousel: Mismo challenge, 10 slides deconstructing belief

**Implicación:** Si eliges `style=Controversial` para un script, ambos sistemas lo ejecutarán de forma coherente.

---

### 6 Narrative Patterns (identical)

Ambos sistemas usan los mismos patrones para estructura interna:

| Patrón | Estructura | Motor | Cuándo usar |
|--------|-----------|-------|------------|
| **Contraste** | Belief vs Reality / Before vs After | Impacto | Reframing, problem exposure |
| **Error** | Mistake → Consequence → Correction | Impacto | Attacking habits/mindset |
| **Punto de Vista** | Reframe the situation | Aspiración/Educación | Teaching, shifting perspective |
| **Confesión** | Naming the pattern, self-recognition | Reflejo | Validation, taboo naming |
| **Hábito** | Trigger → Behavior → Reward | Aspiración | Before/after transformation |
| **Lista** | Multiple causes, factors, or steps | Educación | Complex problems |

**Uso:** El carousel tiene UN patrón narrativo consistente de slide 1-10. Los scripts también. Pueden ser diferentes entre contenido, pero cada pieza respeta un patrón.

---

### Pain vs Desire Ratio (identical matrix)

Ambos sistemas analizan el contenido con esta matriz:

**What dominates your content?**
- ✓ Pain-heavy (70% attacking problems, fears, mistakes)
- ✓ Desire-heavy (70% promising transformation, revealing opportunities)
- ✓ Balanced (50/50)

**Esto determina:**
- Emotional weight del contenido
- Qué segments del audience resonarán más
- Tone (sharper si pain-heavy, aspirational si desire-heavy)

**Implicación:** Si tu B-Roll script es 70% pain + 30% desire, el carousel derivado tendrá el mismo ratio.

## 3. VALIDATION FRAMEWORK COMPARTIDO

### 11-Element Validation (identical checklist)

**Todos los contenidos (scripts y carousels) deben pasar:**

| Elemento | Qué es | En Scripts | En Carousels |
|----------|--------|-----------|-------------|
| **Gancho** | Opening provocador | Hook es gancho (Slide 1 en carousel, apertura en script) | Slide 1: máx 5 palabras |
| **Patrón Narrativo** | Estructura story (Contraste/Error/POV/etc) | Estructura completa de script | Estructura de 10 slides |
| **Objetivo Claro** | UN objetivo (vender/educar/reposicionar) | Script tiene 1 objetivo | Carousel tiene 1 objetivo |
| **Contenido Relevante** | Conectado con input, sin diversiones | Basado en topic/input | Basado en topic/input |
| **Emoción Activada** | Miedo/Curiosidad/Identificación/Urgencia/Inspiración presentes | Emociones claras en script | Emociones claras en carousel |
| **Caption Estratégico** | Si tiene caption, refuerza gancho + tiene CTA | Não se aplica directamente | Caption refuerza + CTA claro |
| **CTA Poderosa** | Pregunta/Acción incómoda, fácil de responder | Script termina con pregunta/CTA | Slide 10 es pregunta poderosa |
| **Estructura Correcta** | Sigue el patrón prometido | Script tiene ritmo correcto | 10 slides con ritmo correcto |
| **Promesa Cumplida** | Si prometes algo, lo entregas | Script delivery en duración correcta | Carousel delivery en slides 3-9 |
| **Coherencia Total** | Cada parte conecta, tono consistente, sin saltos | Script es coherente de principio a fin | Carousel es coherente slide 1-10 |
| **Valor Real** | Enseña, transforma, o reposiciona—no motivación vacía | Script enseña algo real | Carousel enseña algo real |

**Nada se entrega sin pasar los 11 checks en AMBOS sistemas.**

## 4. TONO & VOZ (IDENTICAL)

### Esto sí lo hacemos:

✓ **Directness** — Sin fluff, sin corporate speak, sin filler
✓ **Specificity** — Principios generales atados a ejemplos concretos
✓ **Tension** — Cada pieza tiene peso emocional (miedo, curiosidad, esperanza)
✓ **Rhythm** — Frases cortas, length variada, momentum
✓ **Psychology** — Hablamos a una persona específica, no a "todos"
✓ **Authenticity** — Suena humano, no algoritmo
✓ **Provocative** — Dispuesto a ser incómodo, nombrar verdades duras

### Esto NO lo hacemos:

✗ Motivación genérica ("¡Puedes lograrlo!")
✗ Corporate speak ("Sinergizar soluciones")
✗ Clickbait sin sustancia ("Los expertos odian este truco")
✗ Vaguedad ("Mejora tus habilidades")
✗ Generic CTAs ("Sigue para más")
✗ Inspiración hueca

### Voice Examples

**B-Roll + Carousels speak like this:**

```
"Tu problema no es vender. Es que nadie entiende por qué escucharte."
"Cuando no comunicas diferente, compites por precio."
"¿Dónde estás perdiendo clientes sin darte cuenta?"
"Si educas sin ofrecer, probablemente también regalas tu experiencia."
"Una sola métrica que ignoras determina si vendes o no: la claridad de quién es tu cliente ideal."
```

**NOT like this:**

```
"Mejorar tus habilidades de venta es importante."
"¡Aprende este secreto que los expertos no quieren que sepas!"
"Sigue estos 5 pasos para tener éxito."
"Estamos aquí para ayudarte a crecer."
```

## 5. PERFIL DE USUARIO → ESTILO MATRIZ (identical)

**Ambos sistemas usan la misma matriz para inferir style:**

| Perfil | Motor Viral | Estilo | Razón |
|--------|-----------|--------|-------|
| **Coach** | Transformación + Impacto | **NEGATIVO** | Attack mindset/error, show correction |
| **Vendedor** | Educación + Reflejo | **INFO SECRETA** | Reveal what competitors hide, reframe |
| **Educador** | Educación + Aspiración | **INFO SECRETA** | Teach what others skip, authority |
| **Creador** | Aspiración + Reflejo | **CONTROVERSIAL** | Challenge beliefs, stand out |
| **Experto** | Impacto + Educación | **CONTROVERSIAL** | Attack conventional wisdom |

**Si el usuario no especifica perfil, AMBOS sistemas infieren del contenido.**

## 6. BATCH GENERATION (identical distribution)

### `--batch=5`

Ambos sistemas generan 5 piezas con esta distribución:
- 1 × Info Secreta
- 1 × Negativo  
- 1 × Controversial
- 2 × variaciones del approach más fuerte

**Variedad garantizada, sin repetición.**

### `--batch=10`

Ambos sistemas generan 10 piezas con esta distribución: **40% Info Secreta / 35% Negativo / 25% Controversial**

- 4 × Info Secreta (highest engagement/saves)
- 4 × Negativo (strong transformation appeal)
- 2 × Controversial (differentiation)

**Cada pieza tiene:**
- Hook diferente (10 fórmulas se distribuyen)
- Patrón narrativo variado
- Emotional emphasis diferente
- Caption & CTA tailored

## 7. CÓMO USARLOS JUNTOS

### Flujo 1: Scripts Only
```
/content-plan "Posicionamiento de marca" --batch=10 --export=docx
→ Output: 10 scripts para reels/videos, validados ✓
```

### Flujo 2: Carousels Only
```
[Abre Sistema Carruseles GPT]
"[Transcripción del video sobre copywriting]" --batch=5 --export=notion
→ Output: 5 carousels, validados ✓
```

### Flujo 3: Integrated (Scripts + Carousels)
```
/content-plan "Personal branding" --batch=5 --include-carousels=yes --export=both

→ B-Roll genera 5 scripts
→ Invoca Sistema Carruseles para cada script
→ Total output: 5 scripts + 15 carousels = 20 content pieces para 30 días
→ Todos validados, todos exportados a DOCX + Notion
```

### Flujo 4: Standalone Carousels from Article
```
[Abre Sistema Carruseles GPT]
"[Artículo sobre email marketing]" --batch=10 --export=both
→ Output: 10 carousels desde un solo artículo, múltiples ángulos
→ Mismo framework psicológico, distinto medium
```

## 8. COHERENCIA CHECKLIST

**Cuando uses ambos sistemas, verifica:**

- ✓ Ambos usan las mismas 10 fórmulas virales
- ✓ Ambos respetan los mismos 3 estilos de ejecución
- ✓ Ambos analizan con los mismos 4 motores virales
- ✓ Ambos validación contra los mismos 11 elementos
- ✓ Ambos mantienen el mismo tono/voz
- ✓ Ambos aplican la misma pain/desire ratio
- ✓ Ambos usan los mismos 6 patrones narrativos
- ✓ Ambos salen con la misma temática de marca

**Si algo no encaja, probablemente uno de los sistemas no está configurado correctamente.**

## 9. CUANDO UPGRADEAR

**Si descubres en tus datos que:**
- "Los carousels con hook 'Resultado' generan 40% más saves"
- "Mi audiencia es 80% Info Secreta, 20% Negativo, 0% Controversial"
- "El emotional motor 'Impacto' genera 3x conversiones"

**Comparte esto con ambos sistemas.** Ambos pueden recalibrarse juntos manteniendo coherencia.
